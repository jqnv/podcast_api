# Podcast catalog

podcast_api is a Python API to get JSON info and display it for the user.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install podcast_api.

```bash
pip install virtualenv
pip install flask
pip install requests
```
